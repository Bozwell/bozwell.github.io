python argparse

2019-11-27

자주 반복하는 업무를 효율적으로 처리하기 위해 커맨드 라인 툴을 만들고 있다.

먼저 python 으로 만들어 보고 있는데, argparse 공식문서를 보니, 내가 게으른 것인지 진도가 잘 나가지 않았다.

세부적인 것은 나중에 필요할 때 공식문서를 참고 하더라도, 퀵하게 보고 따라할 수 있는 예제가 있으면 좋겠다는 생각으로 찾아보니 대부분의 검색되는 포스트들이 공식문서를 보고 정리한 정도의 문서여서 영어로 검색해 보니 괜찮은 문서를 발견했다.

예제

store - 인자값을 저장한다. 위의 예제에서는 -s 뒤에 오는 값은 dest(simple_value)에 저장된다.

store_const - const에 지정된 값(value-to-store)이 dest(constant_value)에 저장된다.

store_true - dest(boolean_switch)에 True를 저장한다.

store_false - dest(boolean_switch)에 False를 저장한다.

append - 인자를 리스트 형태로 저장한다. 옵션을 여러번 지정할 때 유용하다고 한다.

append_const - const에 지정된 값을 리스트형대로 저장한다.
